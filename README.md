# Leetcode-Notebook


leetcode之前零散地刷了200+题，也没有写报告和及时复习，遗忘现象严重，今年决心重新从0题刷起，并且将做题过程中的思考整理成笔记，实现代码也将更加重视注释。

所有的题目均会提供C++和python两个版本的实现代码，后期看是否有空余时间来决定是否增加java实现代码。
